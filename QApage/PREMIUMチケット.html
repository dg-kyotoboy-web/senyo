PREMIUMチケットに関して

Q. PREMIUMチケットの使い方は？併用はできますか？

A. PREMIUMチケットは春・夏・冬に販売されるPREMIUM SELECTIONを購入すると貰えるものになります。
種類に関しましては、都度変更になりますのでご注意ください。

※詳細に関しては、以下参照

① 片道交通費無料

内容

チケット1枚の使用につき、片道の交通費が最大￥1,000まで無料となります。
チケットを2枚使用した場合、往復分の交通費（最大￥2,000まで）が無料となります。
なお、他店呼出（神戸店・なんば店）に関しましては、チケット2枚の使用で往復分の交通費が全額無料となります。

注意事項

※無料上限金額を超えた場合、超過分はお客様負担となります。

併用可能物

KOクーポン / ギフト / イクポン / サービスタイム / メンバーズ特典 / チケット特典

② 貸切コース延長60分無料

内容

チケット1枚の使用で、各種貸切コース（19時間貸切 / 24時間貸切 / 30時間貸切）において、 延長60分を無料で追加することが可能です。

注意事項

※1コースにつきチケット1枚のみ使用可能
※当日の利用については、ボーイのシフト状況により延長できない場合があります

併用可能物

KOクーポン / ギフト / イクポン / メンバーズ特典 / チケット特典

③ 30時間貸切

内容

チケット2枚の使用で、30時間貸切コースをご利用いただけます。

注意事項

※1コースにつきチケット2枚が必要になります
※当日予約の場合、予約状況によってはご利用いただけない場合があります
※開始時間は原則として13:00～25:00の間となります

併用可能物

KOクーポン / ギフト / イクポン / メンバーズ特典 / チケット特典

④ ランキングポイント 20pt

内容

チケット利用月の翌月発表のランキングポイントに20ptが反映されます。

併用可能物

なし

⑤ 新作DVDがもらえる

内容

チケット10枚の使用で、新作DVDを無料で受取が可能になります。

注意事項

※新作のリリース日に注意してください
※4月10日～8月31日まで新作DVDの注文が可能

併用可能物

なし

⑥ ペケナナLiveポイント5,000pt付与

内容

チケット20枚の使用で、ペケナナLiveで使えるペケナナポイント5,000ptをプレゼント。

注意事項

※チケット利用日からポイント付与までに3～5日ほどかかります

併用可能物

なし
